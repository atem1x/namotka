Расчёты намотчика

Все расчёты выполняются прямо в браузере. Результаты — ориентир для практики, окончательное решение всегда за мастером.

🔁 Пересчёт обмотки на другое напряжение

W₂ = W₁·U₂/U₁ — витки меняются пропорционально напряжению. Провод: d₂ = d₁·√(U₁/U₂) — сечение растёт во столько же раз, во сколько упало число витков (заполнение паза сохраняется).

📏 Провод: диаметр → сечение → ток

🔀 Замена провода (подбор эквивалента)

Нет нужного диаметра? Подбери замену из нескольких жил.

Варианты подбираются из стандартного ряда диаметров с отклонением сечения не более ±6%. Две-три тонкие жилы мотать легче, чем одну толстую.

⚡ Ток двигателя

Трёхфазная сеть: I = P / (√3·U·КПД·cosφ). Быстрая прикидка для 380 В: I ≈ 2·P(кВт).

🔋 Конденсатор для однофазного включения

Cраб = 4800·I/U (Δ) или 2800·I/U (Y). Пусковой — в 2,5–3 раза больше рабочего, включается через кнопку на 2–3 секунды. Конденсаторы — плёночные (МБГО, МБГЧ, CBB60) на напряжение не ниже 450 В.

🧵 Сопротивление обмотки

R = ρ·L/S, медь ρ = 0,0175 Ом·мм²/м при 20 °C, температурный коэффициент 0,004/°C. Сравни расчёт с замером мультиметром — фазы не должны отличаться более чем на 2–5%.

🧮 Помощник по пазам (q, τ, шаг)

q — пазов на полюс и фазу, τ — полюсное деление. Развёртку обмотки для своих Z и 2p смотри в разделе «Схемы».

📐 Оценка мощности по сердечнику

Табличка сгорела? Прикинь мощность по железу статора.

Грубая эмпирическая оценка P ≈ k·D²·l·n·10⁻⁶ (D, l в см). Точность ±25% — дальше сверяйся с таблицами по числу пазов и габариту.

Схемы соединений и обмоток

Клеммная колодка, звезда/треугольник, конденсаторный пуск и генератор развёртки обмотки.

⭐ Звезда (Y)

Перемычки соединяют весь верхний ряд: W2–U2–V2. Сеть подаётся на U1, V1, W1. Для двигателя 220/380 В звезда — это включение в сеть 380 В. Фазное напряжение обмотки = линейное/√3.

△ Треугольник (Δ)

Перемычки ставятся вертикально: W2–U1, U2–V1, V2–W1. Для двигателя 220/380 В треугольник — сеть 220 В (или 380 В для двигателя 380/660). На каждую обмотку приходится полное линейное напряжение.

ℹ Как выбрать схему

На щитке написано два напряжения, например 220/380 В Δ/Y: первая цифра — для треугольника, вторая — для звезды. Правило простое: на меньшее напряжение — треугольник, на большее — звезда. Ошибёшься в меньшую сторону — двигатель не выдаст момент; в большую — сгорит обмотка.

🔋 Включение в 220 В через конденсатор

Двигатель собирается в треугольник (для 220/380 В). Рабочий конденсатор Cр включён постоянно, пусковой Cп — параллельно ему через кнопку без фиксации, только на время разгона (2–3 с). Реверс — переброс конденсаторной точки на другой сетевой провод.

📊 Подбор ёмкости

P, кВтCраб ~мкФ (Δ, 220В)Cпуск ~мкФ

Ориентир: ~70 мкФ на 1 кВт (треугольник, 220 В). Точный подбор — по нагреву и току: в разделе «Расчёты» есть калькулятор. Потеря мощности при конденсаторном включении — до 30–40%.

🌀 Генератор развёртки однослойной обмотки

Фаза A (U)Фаза B (V)Фаза C (W)

Строится классическая однослойная концентрическая обмотка (при чётном q) или упрощённая равнокатушечная развёртка (при нечётном q). Пунктир — катушка, уходящая через край развёртки. Порядок соединения катушечных групп внутри фазы — «конец с концом, начало с началом» (встречное включение соседних групп).
