# НАМОТКА — портал намотчика электродвигателей

Личный рабочий портал: справочник обмоточных данных АИР, таблицы проводов ПЭТВ, калькуляторы перемотки, схемы соединений и развёртки обмоток, учёт заказов и склада мастерской.

Сайт: https://atem1x.github.io/namotka/

Один файл `index.html` — работает офлайн, данные заказов хранятся в браузере (localStorage), есть экспорт/импорт резервной копии.
