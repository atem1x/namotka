Внутренний Ø статора — типовой для габарита; у 4/6/8-полюсных лист общий, у 2-полюсных — свой, поменьше. Длина сердечника — расчётная оценка (±20%). Обмотка: 1сл. конц. — однослойная концентрическая, 2сл. — двухслойная, «2×…» — два параллельных провода. Расчёты намотчика

Все расчёты выполняются прямо в браузере. Результаты — ориентир для практики, окончательное решение всегда за мастером.

🔁 Пересчёт обмотки на другое напряжение

W₂ = W₁·U₂/U₁ — витки меняются пропорционально напряжению. Провод: d₂ = d₁·√(U₁/U₂) — сечение растёт во столько же раз, во сколько упало число витков (заполнение паза сохраняется).

📏 Провод: диаметр → сечение → ток

🔀 Замена провода (подбор эквивалента)

Подбор эквивалентного сечения из нескольких параллельных проводов.

Варианты подбираются из стандартного ряда диаметров с отклонением сечения не более ±6%. Укладка двух-трёх параллельных проводов меньшего диаметра технологичнее одного провода большого сечения.

⚡ Ток двигателя

Трёхфазная сеть: I = P / (√3·U·КПД·cosφ). Приближённая оценка для 380 В: I ≈ 2·P(кВт).

🔋 Конденсатор для однофазного включения

Cраб = 4800·I/U (Δ) или 2800·I/U (Y). Пусковой — в 2,5–3 раза больше рабочего, включается через кнопку на 2–3 секунды. Конденсаторы — плёночные (МБГО, МБГЧ, CBB60) на напряжение не ниже 450 В.

🧵 Сопротивление обмотки

R = ρ·L/S, медь ρ = 0,0175 Ом·мм²/м при 20 °C, температурный коэффициент 0,004/°C. Сравните расчётное значение с результатом измерения — сопротивления фаз не должны отличаться более чем на 2–5%.

🧮 Помощник по пазам (q, τ, шаг)

q — пазов на полюс и фазу, τ — полюсное деление. Развёртка обмотки для заданных Z₁ и 2p — в разделе «Схемы».

📐 Оценка мощности по сердечнику

Заводская табличка отсутствует? Оценка мощности по размерам сердечника статора.

Грубая эмпирическая оценка P ≈ k·D²·l·n·10⁻⁶ (D, l в см). Точность ±25% — далее уточняйте по таблицам, числу пазов и габариту.

Схемы соединений и обмоток

Клеммная колодка, звезда/треугольник, конденсаторный пуск и генератор развёртки обмотки.

⭐ Звезда (Y)

Перемычки соединяют весь верхний ряд: W2–U2–V2. Сеть подаётся на U1, V1, W1. Для двигателя 220/380 В звезда — это включение в сеть 380 В. Фазное напряжение обмотки = линейное/√3.

△ Треугольник (Δ)

Перемычки ставятся вертикально: W2–U1, U2–V1, V2–W1. Для двигателя 220/380 В треугольник — сеть 220 В (или 380 В для двигателя 380/660). На каждую обмотку приходится полное линейное напряжение.

ℹ Как выбрать схему

На заводской табличке указаны два напряжения, например 220/380 В Δ/Y: первая цифра — для треугольника, вторая — для звезды. Правило простое: на меньшее напряжение — треугольник, на большее — звезда. При включении на пониженное напряжение двигатель не развивает номинальный момент; при включении на повышенное — обмотка выходит из строя.

🔋 Включение в 220 В через конденсатор

Двигатель собирается в треугольник (для 220/380 В). Рабочий конденсатор Cр включён постоянно, пусковой Cп — параллельно ему через кнопку без фиксации, только на время разгона (2–3 с). Реверс — переключение точки подключения конденсатора на другой сетевой провод.

📊 Подбор ёмкости

P, кВтCраб ~мкФ (Δ, 220В)Cпуск ~мкФ

Ориентир: ~70 мкФ на 1 кВт (треугольник, 220 В). Точный подбор — по нагреву и току: в разделе «Расчёты» есть калькулятор. Потеря мощности при конденсаторном включении — до 30–40%.

🧩 Трёхфазные односкоростные: число пазов × частота вращения

По материалам «Справочника обмотчика асинхронных электродвигателей». Отметка — применяемое сочетание числа пазов статора и синхронной частоты вращения (2p = 6000 / n). Нажатие на отметку открывает построение развёртки обмотки для данного сочетания; для части сочетаний (дробное q) однослойная развёртка не строится — выводится предупреждение.

🧩 Однофазные: число пазов × частота вращения

Сочетания приведены справочно. Построение развёрток однофазных обмоток пока не поддерживается. Таблица многоскоростных двигателей будет добавлена по получении продолжения источника.

🌀 Генератор развёртки однослойной обмотки

Фаза A (U)Фаза B (V)Фаза C (W)
